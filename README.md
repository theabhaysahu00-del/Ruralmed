# RuralMed

RuralMed is a comprehensive telemedicine platform designed to connect patients in rural areas with healthcare professionals. It offers a suite of features including role-based portals (Patient, Doctor, Admin, Pharmacy), real-time appointments, secure authentication, and an intuitive dashboard.

## Key Features

- **Role-Based Portals:** Dedicated interfaces and workflows for Patients, Doctors, Administrators, and Pharmacies.
- **Secure Authentication:** JWT-based authentication with role-based access control and robust security measures.
- **Doctor Verification System:** End-to-end doctor onboarding and verification workflow with real-time status tracking.
- **Real-Time Synchronisation:** Live dashboard updates and appointment scheduling.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Zustand, Framer Motion, Recharts
- **Backend:** Node.js, Express, Mongoose, Socket.io
- **Database:** MongoDB

## Getting Started

### Prerequisites
- Node.js installed
- MongoDB installed and running

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/theabhaysahu00-del/Ruralmed.git
   ```

2. Install dependencies for all parts of the application:
   ```bash
   npm run install:all
   ```

3. Start the development server (client and server concurrently):
   ```bash
   npm run dev
   ```

The application will be available at the specified local ports.
